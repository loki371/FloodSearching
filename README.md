# README

## Steps:
1. Install MongoDB
2. Create Database: 
```
use cuu_nan
```
3. Create Table registration_image and road_point
```
db.createCollection("registration_image")
db.createCollection("road_point")
```
4. Start mongoDB
```
mongo
```
5. Start Server
```
./run.sh
```
